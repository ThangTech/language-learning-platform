# Copilot Commit Message Instructions

## Format

Mỗi commit message phải gồm **3 phần**:

```
<emoji> <type>(<scope>): <short summary>

<bullet list mô tả chi tiết các thay đổi>

<footer nếu có breaking change hoặc issue ref>
```

---

## Phần 1 — Dòng tiêu đề (Subject line)

- Bắt đầu bằng emoji + type + scope + summary ngắn
- Tối đa 72 ký tự
- Viết theo kiểu: **động từ + đối tượng** (tiếng Anh hoặc tiếng Việt đều được)
- Không kết thúc bằng dấu chấm

**Format:** `<emoji> <type>(<scope>): <summary>`

**Ví dụ:**
```
🎉 feat(ui): redesign HomePage with modular section components
🔨 refactor(home): split HomePage into 7 focused sub-components
🐛 fix(hero): fix stale closure bug in HeroSlider useEffect
✨ enhance(blog): add category filter and gradient card design
```

---

## Phần 2 — Body (chi tiết thay đổi)

**Bắt buộc** nếu có nhiều hơn 1 thay đổi. Dùng bullet list, liệt kê **từng file hoặc từng chức năng** đã thay đổi.

Mỗi dòng bắt đầu bằng `-` và mô tả **cái gì đã làm** + **lý do hoặc tác động**.

**Ví dụ tốt:**
```
- Add HeroSlider component with auto-play (5s), fade animation, dot/bar indicators
- Add StatsStrip component: 4 key metrics with border-right divider
- Add AboutSection: 2-column layout image + text + CTA
- Add ServicesSection: 3x2 grid with hover underline bar animation
- Add FaqSection: split layout (image left, blue accordion right)
- Add DoctorsSection: 4-column doctor cards with gradient initials avatar
- Add BlogPreviewSection: 3 posts with color-coded gradient by category
- Refactor HomePage.tsx to import/compose all section components (16 lines total)
- Create BlogPage.tsx with category filter tabs and 6 mock articles
- Add /blog route to main.tsx router
- Add Blog nav item in header.tsx between Dịch vụ and Liên hệ
- Generate hero1.jpg, hero2.jpg as slideshow images for HeroSlider
```

---

## Phần 3 — Footer (tùy chọn)

Dùng khi:
- Có **breaking change**: `BREAKING CHANGE: <mô tả>`
- Tham chiếu ticket/issue: `Refs: #42`

---

## Emoji Categories

| Emoji | Type | Khi nào dùng |
|:---:|:---|:---|
| 🎉 | feat | Tính năng mới hoàn chỉnh |
| ✨ | enhance | Cải tiến tính năng đã có |
| 🐛 | fix | Sửa bug |
| 🔨 | refactor | Tái cấu trúc code, không thêm tính năng |
| 💄 | style | Thay đổi UI, CSS, layout thuần thẩm mỹ |
| 🧹 | chore | Cập nhật deps, config, tooling |
| 📜 | docs | Tài liệu, README, comments |
| ✅ | test | Thêm hoặc sửa test |
| 🚧 | wip | Work in progress, chưa hoàn thiện |
| 🔒 | security | Sửa lỗ hổng bảo mật, phân quyền |
| ⚡ | perf | Tối ưu hiệu năng |
| 🗑️ | remove | Xóa file, code, dependency không dùng |

---

## Scope (phạm vi)

Scope mô tả module/layer bị ảnh hưởng. Ví dụ:

| Scope | Ý nghĩa |
|:---|:---|
| `ui` | Toàn bộ giao diện chung |
| `home` | Trang chủ / home components |
| `blog` | Blog page và components |
| `auth` | Đăng nhập, phân quyền |
| `dashboard` | Giao diện quản trị |
| `patient` | Module bệnh nhân |
| `admission` | Module nhập viện |
| `invoice` | Hóa đơn, viện phí |
| `api` | Service gọi API |
| `router` | Routing, navigation |
| `config` | Cấu hình build, env, deps |

## Ví dụ commit message hoàn chỉnh

```
🎉 feat(home): redesign HomePage with slideshow hero and modular sections

- Implement HeroSlider with 3 slides, auto-play 5s, fade + slide-up animation
- Add dot indicators and side bar indicators for manual navigation

- Add ServicesSection: 6 specialties in 3x2 grid, animated underline on hover
- Add FaqSection: split layout with blue accordion and image on left
- Add DoctorsSection: 4 cards with gradient initials avatar (no icons)
- Add BlogPreviewSection: 3 posts, color-coded gradient banner by category
- Refactor HomePage.tsx to 16-line composition file
- Add BlogPage.tsx with category filter (6 categories, 6 mock posts)
- Add Blog nav item to header between Dịch vụ and Liên hệ
- Register /blog route in main.tsx
- Generate and add hero1.jpg, hero2.jpg to assets for slideshow
```

## Quy tắc quan trọng

1. **Không viết message chỉ 1 dòng** nếu làm nhiều thứ trong 1 commit.
2. **Mỗi bullet = 1 file hoặc 1 tính năng cụ thể** — không gộp chung mơ hồ.
3. **Không dùng "misc", "various", "update things"** — phải cụ thể.
4. Nếu chỉ 1 thay đổi nhỏ thì OK với 1 dòng tiêu đề.
5. Body phân tách với tiêu đề bằng **1 dòng trống**.
